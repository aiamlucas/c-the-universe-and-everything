### Small Subset of My Neovim Keybindings

The idea here is to **rotate and practice a small set of keybindings** over time,  
so we can build muscle memory gradually.

---

# Tabs

| Keybinding            | Action            | Description            |
|----------------------|--------------------|------------------------|
| `<Space><Tab><Tab>`  | `:tabnew`          | New tab                |
| `<Space><Tab>d`      | `:tabclose`        | Close current tab      |
| `<Space><Tab>o`      | `:tabonly`         | Close other tabs       |
| `<Space><Tab>[`      | `:tabprevious`     | Previous tab           |
| `<Space><Tab>]`      | `:tabnext`         | Next tab               |
| `<Space><Tab>f`      | `:tabfirst`        | First tab              |
| `<Space><Tab>l`      | `:tablast`         | Last tab               |

---

# Terminal

| Keybinding     | Description              |
|----------------|--------------------------|
| `<Space>ft`    | Terminal (root dir)      |
| `<Space>fT`    | Terminal (current dir)   |
